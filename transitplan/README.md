# transitplan

A custom OOP Python library for accessible public transit trip planning.

## Features

- Route modeling with stops, times, and connections
- Trip planning algorithm (shortest path, fewest transfers)
- Accessibility scoring (wheelchair, audio, visual aids)
- Schedule time calculations
- Fare estimation
- Transfer point detection

## Installation

```bash
pip install -e .
```
